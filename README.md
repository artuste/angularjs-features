<h1>Base AngularJS Application</h1>

<p>In command-line go to the project folder.</p>

<h2>Install npm</h2>
<pre>npm install</pre>

<h2>Install bower</h2>
<p>Because Bower is using Git repository, run it in Git bash command-line.</p>
<pre>bower install</pre>

<h2>Run default Grunt task</h2>
<pre>grunt</pre>
